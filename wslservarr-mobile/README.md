# WSLServarr Mobile (React Native)

Expo-based mobile app for the WSLServarr API.

## Run

1. Install dependencies
   - `npm install`
2. Start Expo
   - `npm run start`

Set API base URL in the app to your WSLServarr UI host, e.g.:
- `http://<WSL-or-host-IP>:5055`

`localhost` only works when emulator/device networking maps to the same host.
